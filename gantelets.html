Les gantelets :

Les gantelets mandalorien sont des pièces importantes de l'armure. Ils peuvent gérer d'autres parties de l'armure et/ou être équipés d'armes.
Mes gantelets

Réalisation :
• Le gantelet droit a été réalisé avec la méthode du pepakura en utilisant du carton plume qui est solidifié avec de la fibre de verre.
Puis une couche de mastic a été ajouté pour améliorer les formes.
• Alors que la structure du gantelet gauche est faite en grillage qui a été entouré directement sur le bras.
Après, comme pour l'autre, une couche de vibre de verre puis de mastic.

Le rangefinder :

Le 1er levier permet de monter/descendre mon rangefinder.



La lampe :

Le 4ème levier permet d'allumer ou d'éteindre la lampe.
Et le 5ème permet de choisir si je veux simplement la lampe ou si je veux un des deux messages en morse : "SOS" ou "target detected".



Les darts :

Le 3ème levier permet de choisir si je veux tirer un dart explosif (2 darts rouges) ou un dart empoisonné (4 darts jaunes).
Le bouton rouge à l'arrière du gantelet permet de simuler le tir d'un dart.
Dès que le bouton de tir est appuyé, une led s'éteint en fonction du choix du dart (rouge ou jaune) et se recharge quand tous les darts ont été tiré.



Le laser :

Le 2ème levier permet d'activer le laser.

Les primes :

Ce panel permet de voir des cibles qui ont leur tête de mise à prix.
Les cibles sont aléatoires (pour les espèces, les photos, le nom, la taille, l'âge ou le "alive", "dead" et "dead or alive") et changent tout les 2 jours (donc ce ne sont jamais les mêmes cibles d'affichées).



Le droid espion :

Mon droid espion (qui est situé au-dessus de mon épaule) est équipé d'une petite caméra wifi.
Grâce à mon "datapad", je peux voir en temps réel ce que mon droid voit !



Le casseur de code :

Pouvoir s'infiltrer partout est souvent bien pratique.
Ce panel permet de "casser" n'importe quel code de sécurité.



Le déverouilleur de porte :

Ce panel fonctionne avec le scomp link, il permet de déverouiller les portes.



Le radar :

Mon datapad est également équipé d'un radar.



L'heure :

Pouvoir lire l'heure en toute circonstance peux être très important comme pour assister aux rendez-vous avec les clients !
